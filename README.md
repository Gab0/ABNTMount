# ABNTM

This is intended as a shortcut to the LaTeX ABNT template, helping in managing multiple chapters
and references.

Managing references may be complicated yet, and most of the help ABNTM provides is in that regard.

Currently it only supports Pubmed article metadata, so it is best suited for writing on Biologic/Health sciences.

# Usage

Select any folder to be your working dir. There you should have a file named `Sequence.txt`, formatted like this:

```

TBD


```
